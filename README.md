Scboy自动点赞软件
1. 需要安装一款网络驱动器，比如https://sites.google.com/a/chromium.org/chromedriver/ for Chrome driver
    并把驱动器路径加入到"driver_name.txt"
2. 运行"python -m pip install -U selenium"安装selenium库
3. 在程序中加入登录用户名及密码
